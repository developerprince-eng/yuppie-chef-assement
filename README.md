## Set Up

Requirements

* Postgres 9.6 
*pgAdmin 4.5 or better
* Java8
* Docker (optional)
  
  
  In order to run the Application you need to perform the following steps.
  
  1. Export Global variables
  
  if on linux export 
  
  ```bash
  export PG_DB_USER=postgres
  export PG_DB_PASSWORD=postgres
  export DB_HOST=//localhost
  ```
  
  the password is depends on the password you set up on your postgres db
  
  2. Create Database
  
  The second step is to log in the Postgres using pgAdmin and create a database called **yuppie-chef-reviews**
  
  3. [Install JAVA 8 & JDK 1.8](https://www.oracle.com/java/technologies/downloads/)
  
  4. [Install maven](https://maven.apache.org/install.html)
  
  5. [Install Intellij](https://www.jetbrains.com/help/idea/installation-guide.html) (optional)

  6. Load the project with IDE of your choice (intellij preferred) and run the project.
